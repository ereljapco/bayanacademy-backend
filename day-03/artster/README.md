## Guide

Install dependecies.

```
npm i
```

### Simple Usage

Simple usage:

```
node artster.js
```

### Choose font

Get the list of fonts:

```
node artster.js --fonts
```

Set the font:

```
node artster.js --font='Dancing Font'
```
